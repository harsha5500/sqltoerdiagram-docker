This is a my docker version for the project @royalbhati's cool sqltoerdiagram project found here:
[royalbhati/sqltoerdiagram](https://github.com/royalbhati/sqltoerdiagram#)

## build image

 docker run -p 8080:8080 sqltoerdiagram
