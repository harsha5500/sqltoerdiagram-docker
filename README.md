# Docker container for sqltoerdiagram

This is my Docker version for the project @royalbhati's cool __sqltoerdiagram__ project found here: [royalbhati/sqltoerdiagram](https://github.com/royalbhati/sqltoerdiagram#)

## build image

```Docker
docker compose -t sqltoerdiagram .
docker run -p 8080:8080 sqltoerdiagram
```
